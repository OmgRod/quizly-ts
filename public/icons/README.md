## Quick Start

1. Open `generate-icons.html` in your browser (located in the root directory)
2. Click "Generate All Icons"
3. Download each icon
4. Place them in this directory

Or use a professional icon generator:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

For more details, see `PWA_SETUP.md` in the root directory.
